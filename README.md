# STREETMEET63 Bot

Telegram-бот предложки на `aiogram 3` с аккуратным пользовательским UX и двухшаговой админ-модерацией.

## Что делает бот

- показывает пользователю минималистичный стартовый экран;
- принимает фото, фото с подписью и просто текст;
- сохраняет предложки в SQLite;
- пересылает в админ-чат двумя сообщениями:
  - исходный контент;
  - компактная карточка модерации;
- позволяет админу:
  - публиковать;
  - отклонять;
  - редактировать текст;
  - редактировать подпись;
  - смотреть превью;
  - включать анонимный режим;
  - сбрасывать изменения;
  - быстро подставлять подпись шаблоном;
  - убирать подпись полностью.

## Установка

1. Создайте виртуальное окружение:

```bash
python -m venv .venv
```

2. Активируйте его.

Для PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Создайте `.env` на основе `.env.example`.

## Конфиг

```env
BOT_TOKEN=1234567890:your_bot_token_here
ADMIN_IDS=111111111,222222222
ADMIN_CHAT_ID=-1001234567890
CHANNEL_ID=-1009876543210
DATABASE_PATH=bot.db
SUBMISSION_COOLDOWN_SECONDS=180
```

## Что означает конфиг

- `BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather)
- `ADMIN_IDS` — user id админов через запятую
- `ADMIN_CHAT_ID` — чат модерации
- `CHANNEL_ID` — канал или группа публикации
- `DATABASE_PATH` — путь к SQLite-файлу
- `SUBMISSION_COOLDOWN_SECONDS` — защита от спама

## Как узнать ID

- `ADMIN_IDS` можно узнать через `@userinfobot`
- `/chat_id` покажет `chat_id` и `user_id` администратору в личке или группе
- если бот админ в канале, `/chat_id` в канале покажет `chat_id`

## Пользовательский UX

На `/start` бот показывает:

```text
STREETMEET63 — предложка.
Отправь фото, фото с текстом или просто текст. Мы передадим это в модерацию.
```

Reply-клавиатура:

- `📤 Отправить пост`
- `ℹ️ Как это работает`

После успешной отправки бот отвечает только:

```text
✅ Пост отправлен на рассмотрение
```

## Админский UX

В админ-чат приходит два сообщения:

1. исходный контент без лишнего оформления;
2. компактная карточка со статусом и кнопками.

Карточка содержит:

- номер заявки
- автора
- тип контента
- статус
- время создания
- текущий текст
- текущую подпись

Кнопки модерации:

- `✅ Опубликовать`
- `❌ Отклонить`
- `✏️ Текст`
- `🏷 Подпись`
- `👁 Превью`
- `🙈 Анонимно`
- `♻️ Сбросить`
- `@username`
- `от @username`
- `ID автора`
- `🗑 Убрать подпись`

## Поведение превью

Кнопка `👁 Превью` отправляет отдельное сообщение в админ-чат в том же виде, в каком пост будет опубликован.

## Поведение редактирования

- текст и подпись редактируются через FSM
- после изменения обновляется существующая карточка
- лишние сообщения после успешного обновления не отправляются
- после публикации или отклонения кнопки исчезают
